Les positions en css

Static
Relative
Absolute
Fixed
Sticky

Static : L'élément static reste à la même place qu'initial et reste dans le flux si pas de données. Position par défaut.

Absolute : L'élément se déplace par rapport à la page et sort du transit
en passant par-dessus ce dernier.

Relative : L'élément se place à la suite et reste dans le transit.

Fixed : L'élément reste fixé dans la page à son emplacement donnée en sortant du flux. Il n'est pas sensible au scroll.

Sticky : L'élément reste à sa place sans instructions données. Si instructions données, se déplacera en fonction des données avec le scroll. Reste dans le flux.

! L'ordre et la parenté sont la base du css, bien vérifier la hiérarchisation des éléments !
